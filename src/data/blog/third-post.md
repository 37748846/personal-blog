Markdown 是一种轻量级的标记语言，易于学习和使用。

在 Astro 博客中，我们可以使用 Markdown 和 MDX 来编写文章。

## 目录

  - [Markdown 基本语法](#markdown基本语法)
  - [MDX 支持](#mdx支持)

## Markdown 基本语法

- 标题：`#`, `##`, `###`
- 列表：`-`, `*`, `1.`
- 代码：`\`code\`
- 引用：`>`
- 链接：`[text](url)`

## MDX 支持

Astro 支持在 Markdown 中嵌入 JSX 代码，这使得我们可以在文章中使用 React 组件。
