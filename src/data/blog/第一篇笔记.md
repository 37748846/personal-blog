# 第一篇笔记

不懂什么 hexo

我就通过搜索知道这叫 Font Awesome 图标,通过 fa 图标知道了 node 和 next 主题。
然后从 node.js 开始熟悉这些,踩了很多坑,今天终于建站成功!
我会慢慢总结我的踩坑经验,随手发布,不是专业的,全靠摸索!
