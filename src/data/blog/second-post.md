Astro 是一个现代化的静态网站生成器，非常适合构建博客、文档网站和作品集。

在本文中，我将介绍如何快速搭建一个博客。

## 目录

  - [主要特性](#主要特性)
  - [开始使用](#开始使用)

## 主要特性

- 快速构建
- 支持多种框架
- 简洁的 API
- 优秀的性能

## 开始使用

安装 Astro 并创建项目，然后添加文章即可。

```bash
npm create astro@latest my-blog
```
