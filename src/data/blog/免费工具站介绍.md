# 免费工具站 - 为自己方便，为大家方便

![工具站Logo](/images/lazy-person/logo.svg)

## 我们的宗旨

这个免费工具站的宗旨很简单：

**第一，为自己使用方便。**

**第二，为大家方便。**

就这么简单。

## 为什么做这个工具站？

说实话，一开始只是为了自己方便。

每次需要格式转换、编码解码、时间戳转换的时候，都要去网上找各种工具。有的要注册，有的有广告，有的还要收费。

太麻烦了。

于是就想：为什么不自己做一个呢？

![为什么做工具站](/images/lazy-person/efficiency.jpg)

## 工具站的特点

### 完全免费

所有工具都是免费的，没有隐藏收费，没有广告干扰。

### 无需注册

打开就能用，不需要注册账号，不需要登录。

### 简单易用

界面简洁，操作简单，一看就会。

### 持续更新

我们会不断增加新的工具，满足更多需求。

![持续更新](/images/lazy-person/coding.jpg)

## 现有的工具

### 文本工具

- **Base64 编码/解码** - 快速进行 Base64 编码和解码
- **URL 编码/解码** - 处理 URL 特殊字符
- **JSON 格式化** - 美化 JSON 数据
- **正则表达式测试** - 测试和调试正则表达式

### 图片工具

- **图片压缩** - 压缩图片大小，节省空间
- **图片裁剪** - 裁剪图片到指定尺寸
- **格式转换** - 转换图片格式（PNG/JPG/WebP）

### 开发工具

- **时间戳转换** - 时间戳与日期时间互转
- **颜色转换** - HEX/RGB/HSL 颜色格式转换
- **二维码生成** - 快速生成二维码
