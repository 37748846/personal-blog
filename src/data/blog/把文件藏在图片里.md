如何把文件藏在图片里
====================

你有没有想过把一些重要的文件隐藏在一张普通的图片里，让别人看不出来呢？这其实是一种简单而有效的加密方法，只需要用到一些基本的命令行操作，就可以实现文件和图片的合并和分离。下面就来介绍一下具体的步骤：

## 目录

  - [准备工具](#准备工具)
  - [合并文件和图片](#合并文件和图片)
  - [分离文件和图片](#分离文件和图片)
  - [小结](#小结)

## 准备工具

- 一张图片，最好是 jpg 格式的，这里我们用 1.jpg 作为例子；
- 一个需要隐藏的文件，可以是任何格式的，这里我们用 2.doc 作为例子；
- 一个压缩软件，可以是 WinRAR 或者 7-Zip 等，这里我们用 WinRAR 作为例子；
- 一个命令行窗口，可以按 Win+R 键打开“运行”窗口，输入 cmd 并回车；

## 合并文件和图片

- 首先，我们需要把需要隐藏的文件压缩成一个压缩包，这里我们用 WinRAR 把 2.doc 压缩成 2.zip；
- 然后，我们需要把压缩包和图片放在同一个文件夹里，这里我们用 C:\test 作为例子；
- 接着，我们需要在命令行窗口中切换到 C:\test 目录下，可以输入 cd C:\test 并回车；
- 最后，我们需要输入一个合并命令，格式为 copy /b 图片名+压缩包名 新图片名，并回车。这里我们输入 copy /b 1.jpg+2.zip 3.jpg，并回车。这样就会生成一个新的图片 3.jpg，它看起来和 1.jpg 一样，但是其实包含了 2.zip 的内容。

## 分离文件和图片

- 如果我们想要从 3.jpg 中提取出 2.zip 的内容，只需要把 3.jpg 的后缀名改成 zip，然后用压缩软件打开它就可以了；
- 如果我们想要从 3.jpg 中提取出 1.jpg 的内容，只需要把 3.jpg 复制一份，并把复制的文件改成 1.zip，然后用压缩软件打开它就可以了。注意不要直接改原文件的后缀名，否则会破坏原图片的内容。

## 小结

通过上面的方法，我们就可以实现将文件隐藏在图片中的功能，这样可以保护我们的隐私和安全。当然，这种方法也不是绝对安全的，如果有人怀疑你的图片中有隐藏内容，他们也可以用同样的方法来尝试分离出来。所以，在使用这种方法时，还是要注意不要让别人发现你的秘密哦。
