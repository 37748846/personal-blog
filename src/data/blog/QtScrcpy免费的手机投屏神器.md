## QtScrcpy: 用电脑控制 Android 手机的神器

你是否有过这样的需求，想要在电脑上操作手机，或者在大屏幕上看手机的内容？如果你有，那么 QtScrcpy 是你的最佳选择。QtScrcpy 是一款基于 Scrcpy 开源项目重构的软件，它可以通过 USB 或网络连接 Android 设备，并进行显示和控制。它不需要 root 权限，也不需要在手机上安装任何东西。 它支持 Windows，MacOS 和 Linux 三大主流桌面平台。

### QtScrcpy 的特点

QtScrcpy 有以下几个特点，让它成为了手机投屏控制的神器：

* 精致：仅显示设备屏幕，无多余界面
* 性能：30\~60fps 的帧率，流畅无卡顿
* 质量：支持 1920×1080 以上的分辨率，画面清晰
* 低延迟：通过 USB 连接时，延迟低至 35ms
* 多点触控：支持在电脑上使用鼠标模拟多点触控
* 按键映射：支持在 Windows 下自定义按键映射，方便玩游戏
* 多设备连接：支持同时连接多个 Android 设备，并分屏显示和控制

### QtScrcpy 的使用方法

要使用 QtScrcpy，你只需要按照以下几个步骤：

1. 下载并安装 QtScrcpy 软件，可以从 GitHu 或者 Gitee 上获取最新版本。
2. 打开 QtScrcpy 软件，选择语言为中文，并设置好投屏参数，如分辨率，码率等。
3. 用 USB 线将 Android 设备连接到电脑，并开启 USB 调试模式。
4. 点击 “启动” 按钮，等待几秒钟后，就可以看到手机屏幕出现在电脑上了。
5. 你可以用鼠标和键盘来操作手机，也可以点击 “设置” 按钮来调整投屏参数或者设置按键映射。
6. 如果你想要连接多个设备，你可以点击 “添加设备” 按钮来添加新的设备，并选择不同的窗口布局来分屏显示。

### QtScrcpy 的优势

相比于其他的手机投屏控制软件，QtScrcpy 有以下几个优势：

* 免费开源：QtScrcpy 是完全免费的软件，没有任何广告或者水印。它也是开源的软件，你可以在 GitHub 或者 Gitee 上查看它的源代码，并参与到它的开发中。
* 跨平台：QtScrcpy 支持 Windows，MacOS 和 Linux 三大主流桌面平台，你可以在任何一台电脑上使用它。
* 简单易用：QtScrcpy 不需要 root 权限，也不需要在手机上安装任何东西。你只需要用 USB 线或者网络连接手机和电脑，就可以轻松投屏控制。
* 功能强大：QtScrcpy 不仅可以投屏控制，还可以自定义按键映射，支持多点触控，支持多设备连接，支持分屏显示，支持录屏等功能。
* 性能优异：QtScrcpy 有着高帧率，高分辨率，低延迟的特点，让你在电脑上享受手机的画面和操作。
